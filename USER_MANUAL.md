## For Students

### Dashboard Overview

Your dashboard shows:
- **XP and Level**: Your current experience points and level
- **Current Streak**: Days of continuous learning
- **Completed Quizzes**: This week's progress
- **Achievements**: Badges and milestones
- **Continue Learning**: Resume your courses
- **Quick Actions**: AI Tutor, Practice Quiz

### Using the AI Tutor

1. **Starting a Chat**
   - Click "AI Tutor" from dashboard or navigation
   - Type your question in the chat box
   - Press Enter or click Send

2. **Voice Input** (Optional)
   - Click the microphone icon
   - Speak your question clearly
   - AI will transcribe and respond

3. **Getting Help**
   - Ask subject-specific questions
   - Request explanations of concepts
   - Get step-by-step solutions
   - Practice problems

4. **Tips for Better Results**
   - Be specific in your questions
   - Mention the subject and topic
   - Ask follow-up questions for clarity
   - Use Bangla or English as comfortable

### Taking Quizzes

1. **Finding Quizzes**
   - Go to "Quiz" section
   - Browse by subject or difficulty
   - Or let AI generate custom quiz

2. **Starting a Quiz**
   - Select a quiz
   - Review instructions
   - Click "Start Quiz"
   - Timer begins (if applicable)

3. **Answering Questions**
   - Read each question carefully
   - Select your answer
   - Click "Next" to proceed
   - Use "Previous" to review

4. **Submitting Quiz**
   - Review all answers
   - Click "Submit Quiz"
   - View instant results
   - See correct answers and explanations

5. **Adaptive Difficulty**
   - Quiz difficulty adjusts based on performance
   - Correct answers ‚Üí harder questions
   - Incorrect answers ‚Üí easier questions
   - Personalized learning experience

### Tracking Progress

1. **Progress Dashboard**
   - View subject-wise progress
   - See completion percentages
   - Track time spent learning
   - Monitor quiz scores

2. **XP and Levels**
   - Earn XP for completing activities
   - Level up as you learn more
   - Unlock new features and content

3. **Achievements**
   - Earn badges for milestones
   - Complete challenges
   - View achievement gallery
   - Share accomplishments

4. **Streaks**
   - Maintain daily learning streak
   - Get streak freeze power-ups
   - Compete on leaderboards

### Offline Learning

1. **Downloading Content**
   - Go to Settings ‚Üí Offline Access
   - Select subjects to download
   - Wait for download to complete
   - Content available offline

2. **Using Offline**
   - Access downloaded quizzes

## Troubleshooting

### Login Issues

**Problem**: Can't log in
- **Solution**: 
  - Check email and password
  - Use "Forgot Password" link
  - Clear browser cache
  - Try different browser

**Problem**: Account locked
- **Solution**:
  - Wait 15 minutes
  - Contact support
  - Reset password

### Quiz Problems

**Problem**: Quiz won't load
- **Solution**:
  - Refresh page
  - Check internet connection
  - Clear browser cache
  - Try different browser

**Problem**: Lost progress during quiz
- **Solution**:
  - Platform auto-saves answers
  - Resume from last question
  - Contact support if issue persists

### AI Tutor Issues

**Problem**: AI not responding
- **Solution**:
  - Check internet connection
  - Refresh page
  - Try rephrasing question
  - Report if problem continues

**Problem**: Voice input not working
- **Solution**:
  - Allow microphone permission
  - Check microphone settings
  - Use text input instead
  - Try different browser

### Offline Mode Issues

**Problem**: Content not downloading
- **Solution**:
  - Check internet connection
  - Ensure sufficient storage
  - Try downloading smaller chunks
  - Clear app cache

**Problem**: Sync not working
- **Solution**:
  - Check internet connection
  - Force manual sync
  - Resolve conflicts
  - Contact support

### Performance Issues

**Problem**: Platform running slow
- **Solution**:
  - Close other tabs/apps
  - Clear browser cache
  - Update browser
  - Check internet speed

**Problem**: Videos not playing
- **Solution**:
  - Check internet speed
  - Update browser
  - Enable JavaScript
  - Try different browser

## FAQ

### General Questions

**Q: Is ShikkhaSathi free?**
A: Yes, basic features are free. Premium features may require subscription.

**Q: What grades does it support?**
A: Classes 6-12 for both Bangla and English medium.

**Q: Can I use it on mobile?**
A: Yes, it works on all devices - desktop, tablet, and smartphone.

**Q: Is internet required?**
A: Internet needed for initial setup and AI features. Offline mode available for quizzes and materials.

**Q: How is my data protected?**
A: We use industry-standard encryption and never share personal data.

### Student Questions

**Q: How do I improve my level?**
A: Complete quizzes, maintain streaks, and engage with content regularly.

**Q: Can I retake quizzes?**
A: Yes, most quizzes can be retaken for practice.

**Q: How does adaptive learning work?**
A: Platform adjusts difficulty based on your performance automatically.

**Q: Can I study multiple subjects?**
A: Yes, access all subjects included in your grade.

### Teacher Questions

**Q: How many students can I add?**
A: No limit on student count in your classes.

**Q: Can I create custom assessments?**
A: Yes, create fully customized assessments with various question types.

**Q: How do I track student progress?**
A: Use the analytics dashboard for detailed insights.

**Q: Can I import existing questions?**
A: Yes, import from Excel, CSV, or use our question bank.

### Parent Questions

**Q: How do I monitor my child?**
A: Access parent dashboard for comprehensive progress reports.

**Q: Can I set learning goals?**
A: Yes, set daily/weekly targets and track achievement.

**Q: How do I contact teachers?**
A: Use the messaging feature in parent portal.

**Q: Are there parental controls?**
A: Yes, set screen time limits and content restrictions.

### Technical Questions

**Q: Which browsers are supported?**
A: Chrome, Firefox, Safari, Edge (latest versions).

**Q: How much storage is needed?**
A: Minimum 100MB for offline content.

**Q: Can I use it on multiple devices?**
A: Yes, log in from any device with your credentials.

**Q: Is there a mobile app?**
A: Currently web-based, but works great on mobile browsers. Native app coming soon.
